一个异环抽卡模拟器。基于网上二测和三测信息编写的。
需要C++20，几乎全中文编程。
可以用于统计出金分布、平均每金净消耗多少抽等。

叠甲：本人没参加过异环内测，一切卡池消息均来自互联网。
     尤其是很难找到完整的卡池规则展示。
     如有错误，请指正。
	   例如：A级保底机制不是很完整。不排除地图部分区域错误。

eg：某次模拟中，一共抽到了1000000金；
平均抽数53.2672，平均净抽数48.5813；
捉到宝箱守护者：473931次；
放走宝箱守护者：1292184次。

常规岛1：164004；
常规岛2：26211；
概率提升主区域：403103；
概率提升岛1：48691；
概率提升岛2：45826；
